get_leaf_type(type_)#

Get the definition of the input type, bypassing the aliases.

See the get_leaf_type example.

Parameters:
type_suite.Type

Input type.

Returns:
suite.Type

Definition of the input type or the input type itself itself if it is not an alias, such as an instance of NamedType or a predefined type. It is not a named type unless it is predefined.
