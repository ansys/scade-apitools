NAryOp#

class NAryOp(expression)#

Bases: AryOp

Provides an expression with two or more operands.

The format is <operand> <operator> <operand> <operator> ... <operator> <operand>, where <operator> is one of these:

  • @

  • and, or, xor

  • +, *

  • land, lor

See the n_ary_op example.

Parameters:
expression

Call expression to wrap.

Notes

The xor operator is binary, but it is stored as a nary operator in the XSCADE files.
