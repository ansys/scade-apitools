



CallExpression
==============

.. py:class:: CallExpression(expression)
   :canonical: ansys.scade.apitools.expr.access.CallExpression



Bases: :py:obj:`Expression`


   Provides the abstract class for expression calls.

   There are three kinds of operator calls:

   * Predefined operators
   * User operators
   * Higher-order operators

   :Parameters:

       **expression**
           Call expression to wrap.
