ansys.scade.apitools.create.declaration.create_graphical_operator#

create_graphical_operator(owner, name, path, visibility=VK.PUBLIC, symbol_file=None, state=False)#

Create an instance of Operator with a graphical diagram.

An operator shall have a name.

Parameters:

  • owner (suite.Package) – Owner of the operator, either the model itself or a package.

  • name (str) – Name of the operator.

  • path (Path) –

    Path of the file to store the oprator.

    This parameter is optional if the package’s owner is a package. When path is None and owner is the model, the operator is stored in the model’s default file.

  • visibility (VK) – Accessibility of the operator, either public or private.

  • symbol_file (Path) – Optional Path of the file defining the symbol of the operator.

  • state (bool) – True if the operator is a node.

Return type:

suite.Operator
